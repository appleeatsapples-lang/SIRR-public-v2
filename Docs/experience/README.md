# Experience — SIRR Showcase

This directory will host a JSX visual rendering of the engine's synthetic
demo profile.

The locked v12.1 JSX showcase that demonstrates the engine's output
grammar against a worked example was personal-data-bound and was
removed from the public repo. A clean synthetic version (rendering
FATIMA AHMED OMAR ALKATIB / 1990-03-15 / Cairo from
`Engine/fixtures/synthetic_output.json`) is a deferred Codex deliverable.

For the live, machine-rendered demo, see:
- `https://web-production-ec2871.up.railway.app/view/demo`
- Or run `Engine/web_backend/server.py` locally and hit `/view/demo`

The JSON shape that any future JSX consumer should bind to is
`Engine/fixtures/synthetic_output.json` (1.04 MB, 238 modules + synthesis
+ semantic_reading + psychological_mirror).
